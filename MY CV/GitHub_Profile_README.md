### 🌟 Featured Systems & Projects

#### 🐝 [ResumAI](https://github.com/SantanDon/ResumAI)
An industrial-grade CV building and outreach ecosystem that solves AI text hallucination. By leveraging a **5-worker Swarm Engine** that votes on every decision, ResumAI achieves fault-tolerant output from small, local 1B parameter models.
*   **Tech Stack**: TypeScript, React, Express, SQLite, Puppeteer, Ollama.
*   *Key Achievement*: Built a custom orchestrator running a delta-change synchronization engine and ATS-optimized, Harvard-style PDF rendering.

#### ⚖️ [DocketDive](https://github.com/SantanDon/DocketDive)
South Africa’s first AI-native legal research assistant. DocketDive uses a multi-stage RAG pipeline to search and index South African case law, helping legal practitioners find, analyze, and cite precedent with zero hallucinations.
*   **Tech Stack**: Next.js 14, TypeScript, LangChain, Vector DB, Ollama.

#### 🎧 [StudyPodLM](https://github.com/SantanDon/StudyPodLM)
A multi-modal AI learning workspace that converts textbook PDFs, video links, and audio into structured study guides, complete with a Groq-accelerated text-to-speech audio player.
*   **Tech Stack**: React, Vite, Groq AI, Tesseract OCR, Web Audio API.
